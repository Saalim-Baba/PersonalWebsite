## Getting Started 🛠

To run this project locally:

1. Clone the repository:  
   ```bash
   git clone https://github.com/Saalim-Baba/PersonalWebsite.git
   ```
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Start the development server:  
   ```bash
   npm run dev
   ```
